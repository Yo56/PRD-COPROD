It must have a name and optionally any of\+: alignment, border, fill, font, number format, protection. A format in xlnt corresponds to the alignment, border, fill, font, number format, and protection settings applied to a cell via right-\/click-\/$>$\char`\"{}\+Format Cells\char`\"{}.

\subsubsection*{template$<$typename T, ssize\+\_\+t Dims$>$\newline
class pybind11\+::detail\+::unchecked\+\_\+reference$<$ T, Dims $>$}

Proxy class providing unsafe, unchecked const access to array data. This is constructed through the {\ttfamily unchecked$<$T, N$>$()} method of {\ttfamily array} or the {\ttfamily unchecked$<$N$>$()} method of {\ttfamily \mbox{\hyperlink{classpybind11_1_1array__t}{array\+\_\+t}}$<$T$>$}. {\ttfamily Dims} will be -\/1 for dimensions determined at runtime.

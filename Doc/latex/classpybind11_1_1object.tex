\doxysubsection{Detailed Description}
\textbackslash{}rst Holds a reference to a Python object (with reference counting)

Like {\ttfamily handle}, the {\ttfamily object} class is a thin wrapper around an arbitrary Python object (i.\+e. a {\ttfamily Py\+Object $\ast$} in Python\textquotesingle{}s C API). In contrast to {\ttfamily handle}, it optionally increases the object\textquotesingle{}s reference count upon construction, and it always$\ast$ decreases the reference count when the {\ttfamily object} instance goes out of scope and is destructed. When using {\ttfamily object} instances consistently, it is much easier to get reference counting right at the first attempt. \textbackslash{}endrst

\doxysubsection{Detailed Description}
Describes a unit of data in a worksheet at a specific coordinate and its associated properties. 

Properties of interest include style, type, value, and address. The Cell class is required to know its value and type, display options, and any other features of an Excel cell.\+Utilities for referencing cells using Excel\textquotesingle{}s \textquotesingle{}A1\textquotesingle{} column/row nomenclature are also provided.

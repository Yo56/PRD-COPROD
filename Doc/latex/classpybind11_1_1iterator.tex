\doxysubsection{Detailed Description}
\textbackslash{}rst Wraps a Python iterator so that it can also be used as a C++ input iterator

Caveat\+: copying an iterator does not (and cannot) clone the internal state of the Python iterable. This also applies to the post-\/increment operator. This iterator should only be used to retrieve the current value using {\ttfamily \mbox{\hyperlink{classpybind11_1_1detail_1_1object__api_a5babe5939b794c3408711be7fde14ee0}{operator$\ast$()}}}. \textbackslash{}endrst

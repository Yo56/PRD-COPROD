xlnt is a modern C++ library for manipulating spreadsheets in memory and reading/writing them from/to XLSX files as described in \href{http://www.ecma-international.org/publications/standards/Ecma-376.htm}{\texttt{ ECMA 376 4th edition}}. The first public release of xlnt version 1.\+0 was on May 10th, 2017. Current work is focused on increasing compatibility, improving performance, and brainstorming future development goals. For a high-\/level summary of what you can do with this library, see \href{https://tfussell.gitbooks.io/xlnt/content/docs/introduction/Features.html}{\texttt{ the feature list}}. Contributions are welcome in the form of pull requests or discussions on \href{https://github.com/tfussell/xlnt/issues}{\texttt{ the repository\textquotesingle{}s Issues page}}.\hypertarget{md__c___users_nouet__documents__cours_5_a__dev__p_r_d__prd_coprod__prd_coprod_libs_xlnt_1_5_0__r_e_a_d_m_e_autotoc_md184}{}\doxysection{Example}\label{md__c___users_nouet__documents__cours_5_a__dev__p_r_d__prd_coprod__prd_coprod_libs_xlnt_1_5_0__r_e_a_d_m_e_autotoc_md184}

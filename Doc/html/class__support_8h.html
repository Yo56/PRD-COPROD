bool  allocate_value 
)

Instance creation function for all pybind11 types. It only allocates space for the C++ object (or multiple objects, for Python-side inheritance from multiple pybind11 types), but doesn't call the constructor – an __init__ function must do that. If allocating value, the instance is registered; otherwise register_instance will need to be called once the value has been assigned.
